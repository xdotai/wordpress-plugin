Easily embed your x.ai calendar pages on your Wordpress site.

== Description ==

This plugin gives you the ability to embed your x.ai calendar page in your Wordpress site so that visitors can schedule meetings with you. Configure settings such as the size and appearance of the embedded calendar.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->x.ai Calendar Embed screen to configure the plugin
1. Add your x.ai calendar page URL and any additional settings you like
1. Once saved, simply add the shortcode [xai-calendar] on any page you want to include a calendar!
